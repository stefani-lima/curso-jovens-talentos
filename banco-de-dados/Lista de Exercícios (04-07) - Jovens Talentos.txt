LISTA DE EXERCÍCIOS 02

1. Realize as seguintes consultas no banco de dados com o auxílio dos operadores
AND, OR, NOT e NULL:
a. Selecione todos os contatos que são do DDD 47 e que são do tipo

SELECT *
   FROM TREINA.TBPESSOACONTATO

   FROM TREINA.TBLOGRADOURO

   FROM TREINA.TBPESSOACONTATO

a. Selecione todos os contatos do tipo “Fixo” e ordene-os de forma crescente
utilizando a coluna referente ao número de telefone;

SELECT *
   FROM TREINA.TBPESSOACONTATO

b. Selecione todos os imóveis de madeira e ordene-os de forma decrescente
em relação ao seu comprimento.

SELECT *
   FROM TREINA.TBIMOVEL
  WHERE IMVDESCRICAO LIKE '%MADEIRA%'
  ORDER BY IMVCOMPRIMENTO DESC;

3. Construa um código SQL que retorne os dados de pessoa, quando o código da
pessoa for entre 8 e 12, desconsiderando pessoas que residam nos logradouros de
4 a 7.

SELECT *
   FROM TREINA.TBPESSOA
  WHERE PESCODIGO BETWEEN 8 AND 12
    AND NOT LOGCODIGO BETWEEN 4 AND 7;

4. Construa as seguintes consultas:
a. Crie uma consulta na tabela de pessoas retornando apenas 5 registros.
Desconsidere os três primeiros;

SELECT *
   FROM TREINA.TBPESSOA
  LIMIT 5
 OFFSET 3;

b. Crie uma consulta que retorne apenas as pessoas que não começam com a
letra ‘P’, ordene o resultado em ordem alfabética apresentando apenas
três registros e ignorando os primeiros dois registros.

SELECT TBPESSOA.PESNOME
   FROM TREINA.TBPESSOA
  WHERE PESNOME NOT LIKE 'P%'
  ORDER BY PESNOME
  LIMIT 3
 OFFSET 2;

b. Realize uma contagem da quantidade de imóveis madeira;

c. Realize uma consulta que retorne a somatória das larguras de todos os

6. Realize as seguintes consultas:
a. Na tabela de imóveis, selecione somente as colunas “imvcodigo”,
“imvdescricao”, “imvlargura” e “imvcomprimento”. Renomeie essas
colunas para “Código”, “Descrição”, “Largura” e “Comprimento”. Ordene os
registros de forma decrescente a partir da coluna de código;

SELECT TBIMOVEL.IMVCODIGO AS "CÓDIGO",
       TBIMOVEL.IMVDESCRICAO AS "DESCRIÇÃO",
       TBIMOVEL.IMVLARGURA AS LARGURA,
       TBIMOVEL.IMVCOMPRIMENTO AS COMPRIMENTO
   FROM TREINA.TBIMOVEL
  ORDER BY IMVCODIGO DESC;

b. Para as colunas “pescodigo” e “pesnome”, selecione somente as pessoas

apenas os quatro primeiros registros.

SELECT TBPESSOA.PESCODIGO AS "CÓDIGO DA PESSOA",
       TBPESSOA.PESNOME AS "NOME COMPLETO"
   FROM TREINA.TBPESSOA
  WHERE PESSEXO = 2
  ORDER BY PESNOME
  LIMIT 4;

7. Faça uma seleção que retorne apenas tipos únicos de logradouro.

       TBLOGRADOURO.LOGTIPO
   FROM TREINA.TBLOGRADOURO

Lista de Exercícios 04

1. Realize uma consulta que retorne apenas o “Nome da Pessoa”, o “Tipo do
Logradouro” e a “Descrição do Logradouro”, relacionando as tabelas com o JOIN.
Apresente apenas os logradouros que estão relacionados a pessoas.

SELECT TBPESSOA.PESNOME,
       TBLOGRADOURO.LOGTIPO,
       TBLOGRADOURO.LOGDESCRICAO
  FROM TREINA.TBPESSOA
 INNER JOIN TREINA.TBLOGRADOURO
    ON TBPESSOA.LOGCODIGO = TBLOGRADOURO.LOGCODIGO

2. Consulte as pessoas e seus respectivos imóveis, exibindo as colunas de nome da
pessoa, código do imóvel e descrição do imóvel, inclusive das pessoas que não
possuem imóvel.

SELECT TBPESSOA.PESNOME,
       TBIMOVEL.IMVCODIGO,
       TBIMOVEL.IMVDESCRICAO
  FROM TREINA.TBPESSOA
  LEFT JOIN TREINA.TBPESSOAIMOVEL
    ON TBPESSOA.PESCODIGO = TBPESSOAIMOVEL.PESCODIGO
  LEFT JOIN TREINA.TBIMOVEL
    ON TBPESSOAIMOVEL.IMVCODIGO = TBIMOVEL.IMVCODIGO

3. Faça uma consulta que traga o nome das pessoas, o tipo de contato e número de
contato, de todas as pessoas que possuem celular com o DDD 47. Ordene pelo
nome das pessoas.

SELECT TBPESSOA.PESNOME,
       TBPESSOACONTATO.CTPDESCRICAO,
       TBPESSOACONTATO.CTPNUMERO
  FROM TREINA.TBPESSOA
 RIGHT JOIN TREINA.TBPESSOACONTATO
    ON TBPESSOA.PESCODIGO = TBPESSOACONTATO.PESCODIGO
 WHERE CTPNUMERO LIKE '47%'
 ORDER BY PESNOME;

4. Faça uma consulta relacionando todas as tabelas, exibindo as colunas: nome da
pessoa, número de contato dela, descrição de seu imóvel, área do imóvel, tipo do
logradouro do imóvel, descrição do logradouro do imóvel. Ordene pelo nome das
pessoas.

SELECT TBPESSOA.PESNOME,
       TBPESSOACONTATO.CTPNUMERO,
       TBIMOVEL.IMVDESCRICAO,
       (IMVLARGURA*IMVCOMPRIMENTO) AS AREA,
       TBLOGRADOURO.LOGTIPO,
       TBLOGRADOURO.LOGDESCRICAO
  FROM TREINA.TBPESSOA
  FULL OUTER JOIN TREINA.TBPESSOACONTATO
    ON TBPESSOA.PESCODIGO = TBPESSOACONTATO.PESCODIGO
  FULL OUTER JOIN TREINA.TBTAXA
    ON TBPESSOA.PESCODIGO = TBTAXA.PESCODIGO
  FULL OUTER JOIN TREINA.TBPESSOAIMOVEL
    ON TBPESSOA.PESCODIGO = TBPESSOAIMOVEL.PESCODIGO
  FULL OUTER JOIN TREINA.TBIMOVEL
    ON TBPESSOAIMOVEL.IMVCODIGO = TBIMOVEL.IMVCODIGO
  FULL OUTER JOIN TREINA.TBLOGRADOURO
    ON TBIMOVEL.LOGCODIGO = TBLOGRADOURO.LOGCODIGO
 ORDER BY PESNOME;

5. Realize uma consulta que retorne o nome das pessoas e a quantidade de contatos
que cada uma possui (se não possuir contatos deve aparecer como 0).

SELECT TBPESSOA.PESNOME,
       COUNT(TBPESSOACONTATO.CTPNUMERO)
  FROM TREINA.TBPESSOA
  LEFT JOIN TREINA.TBPESSOACONTATO
    ON TBPESSOA.PESCODIGO = TBPESSOACONTATO.PESCODIGO
 GROUP BY TBPESSOA.PESNOME
