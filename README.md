# 🐘 Jovens Talentos IPM - Trilha PHP

Repositório criado para armazenar os exercícios, projetos e anotações desenvolvidos durante o programa Jovens Talentos da IPM.

## 📚 Sobre o curso

O programa aborda os principais conceitos do desenvolvimento web, com ênfase em PHP e integração com banco de dados, proporcionando uma base sólida para a construção de aplicações web.

## 🛠️ Conteúdos estudados

- Banco de Dados
- HTML5
- CSS3
- JavaScript
- PHP Básico
- Conexão com Banco de Dados
- PHP Orientado a Objetos (POO)

## 📂 Estrutura do Repositório

```text
📁 banco-de-dados
📁 html-css
📁 javascript
📁 php-basico
📁 php-banco-dados
📁 php-poo
```

Cada pasta contém exercícios, exemplos e atividades realizadas ao longo do curso.

## 🎯 Objetivos

- Aprender os fundamentos do desenvolvimento web.
- Desenvolver aplicações utilizando PHP.
- Trabalhar com bancos de dados relacionais.
- Aplicar conceitos de Programação Orientada a Objetos.
- Criar projetos seguindo boas práticas de programação.
